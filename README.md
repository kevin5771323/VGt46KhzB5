# 项目简介

## 核心功能
- 用户端功能：
  - 首页展示轮播图、分类导航与热门/最新菜品
  - 菜品浏览支持按分类/关键词/价格等筛选与排序
  - 基于协同过滤算法的个性化推荐系统，综合点击、点赞、评分多个维度
  - 关键词模糊搜索菜品信息，支持结果排序与筛选
  - 用户中心管理资料、头像与密码，查看收藏/评论/浏览历史
  - 论坛社区支持发帖、回复、点赞与排序筛选

- 管理端功能：
  - 数据统计展示用户、菜品、评论与活跃度等指标及趋势图表
  - 用户管理支持查询用户、查看详情、调整角色与状态
  - 分类管理支持新增/编辑/排序/启用分类并控制前端展示顺序
  - 菜品管理支持增删改查、上架下架、上传图片与富文本详情
  - 评论管理支持查看评论上下文，隐藏或删除不当评论
  - 论坛管理支持管理帖子与回复，设置置顶/精华/状态
  - 轮播图管理支持配置首页轮播图图片、链接、排序与状态

## 适用场景
- 餐饮平台菜品推荐与展示
- 用户个性化需求匹配
- 餐饮社区交流与互动
- 餐饮数据分析与运营决策支持

## 技术特点
- 技术栈：Python Flask + Vue + MySQL
- 运行环境：Python 3.12、MySQL 8.0
- 开发工具：PyCharm、VSCode、Navicat
- 基于协同过滤算法实现个性化推荐
- 提供完整源代码与文档资料
- 算法推荐时会在控制台输出算法介绍及推荐计算全过程
